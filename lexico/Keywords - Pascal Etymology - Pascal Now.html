Keywords

Abstract

The keywords of a programming language are words which have special, predefined syntactic meanings in a program. In Pascal, the keywords are reserved, and cannot be used as identifiers.  Like identifiers, Pascal keywords are not case sensitive.

The first list contains the keywords of ISO 7185 Standard Pascal.  This is the baseline: the other lists are additions.  Borland Delphi also includes the Borland Turbo Pascal keywords, and Free Pascal also includes the Borland Turbo Pascal and Delphi keywords.

ISO 7185 Standard Pascal Keywords

And Downto If Or Then
Array Else In Packed To
Begin End Label Procedure Type
Case File Mod Program Until
Const For Nil Record Var
Div Function Not Repeat While
Do Goto Of Set With

ISO 10206 Extended Pascal Keywords

In addition to the keywords designated in ISO 7185, ISO 10206 designates additional keywords:

And_then Import Module Otherwise Qualified
Bindable Implementation Only Pow Restricted
Export Interface Or_else Protected Value
